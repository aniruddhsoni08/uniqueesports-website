🆘 TROUBLESHOOTING QUICK LINKS
═════════════════════════════════════════════════════════════════════════════

  ERROR: "Cannot find module 'express'"
  → Solution: Run "npm install" (see QUICKSTART.md)

  ERROR: "Error: connect ECONNREFUSED"
  → Solution: MySQL not running. Start MySQL service
  → See QUICKSTART.md troubleshooting section

  ERROR: "Port 5000 already in use"
  → Solution: Change PORT in .env or stop other service
  → See README.md

  ERROR: "Unknown database 'unique_esports'"
  → Solution: Run "mysql -u root -p < schema.sql"
  → See QUICKSTART.md step 2

  ERROR: "Invalid credentials at login"
  → Solution: Check admin password (default: admin123)
  → See ADMIN_CREDENTIALS.sql to change password
