🗄️ DATABASE SCHEMA:

Tables Created:
  
  1. tournaments
     - id (Primary Key)
     - name, game_type, description
     - start_date, end_date
     - prize_pool, status (upcoming|ongoing|completed)
     - youtube_link
     - created_at, updated_at (timestamps)
  
  2. registrations
     - id (Primary Key)
     - tournament_id (Foreign Key)
     - team_name, email, phone
     - players_count, additional_info
     - created_at (timestamp)
  
  3. admin_users
     - id (Primary Key)
     - username (unique), password_hash
     - email
     - created_at (timestamp)

🔌 API ENDPOINTS (All Functional):

PUBLIC ENDPOINTS:
  ✅ GET  /api/tournaments              - List all tournaments
  ✅ GET  /api/tournaments/:id          - Get single tournament
  ✅ POST /api/register                 - Register for tournament

ADMIN ENDPOINTS (Requires Session):
  ✅ POST   /api/admin/login            - Authenticate admin
  ✅ GET    /api/admin/tournaments      - Get all tournaments (admin view)
  ✅ POST   /api/admin/tournaments      - Create new tournament
  ✅ PUT    /api/admin/tournaments/:id  - Update tournament details
  ✅ DELETE /api/admin/tournaments/:id  - Delete tournament
  ✅ GET    /api/admin/registrations/:id - Get tournament registrations
  ✅ POST   /api/admin/export           - Export to Excel
  ✅ GET    /api/admin/download/:file   - Download Excel file

🚀 GETTING STARTED:

1. Install Dependencies:
   npm install

2. Setup Database:
   mysql -u root -p < schema.sql

3. Configure Environment:
   cp .env.example .env
   (Edit .env with your MySQL credentials)

4. Start Server:
   npm start

5. Access Platform:
   Player Portal:  http://localhost:5000/
   Admin Panel:    http://localhost:5000/admin
   Default Login:  admin / admin123
